# NINJA 2.x

NINJA 2.x is a patch format created by Derrick Sobodash (Cinnamon
Pirate), a successor to NINJA 1.x. Despite the shared name, the two
formats are structurally different and not interchangeable.

slap currently has no writing of its own about this format beyond
whatever is encoded in `src/Slap/NINJA2/`. This folder exists as a
preservation holder for the upstream specification.

## Files

- `upstream/ninja2-filespec20.txt` — NINJA 2.0 file format specification.
- `upstream/ninja2-filespec20-rhdn.txt` — NINJA 2.0 file spec as
  distributed on romhacking.net (may differ slightly from the
  author's version).
- `upstream/ninja2-cliusage.txt` — CLI usage documentation for the
  NINJA 2.0 tool.
- `upstream/ninja2-convroms.txt` — notes on converting between ROM
  variants with NINJA 2.0.
