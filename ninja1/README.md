# NINJA 1.x

NINJA 1.x is a patch format created by Derrick Sobodash (Cinnamon
Pirate), released as a PHP reference implementation. It predates
NINJA 2.x, which is a significantly different format despite the
shared name.

NINJA 1.x and NINJA 2.x are **not compatible** — they are different
formats with different wire structures. They share an author and a
naming lineage, nothing else. Treating them as variants of a single
format is incorrect.

slap currently has no writing of its own about this format. This
folder exists as a preservation holder for the upstream specification
and reference implementation.

## Files

- `upstream/ninja-1.01php.tar.gz` — Derrick Sobodash's NINJA 1.01 PHP
  reference implementation.
- `upstream/ninja1-filespec10.txt` — NINJA 1.0 file format
  specification.
